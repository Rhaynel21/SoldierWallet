# Bank logos

Drop each partner bank's **official logo** here as a PNG (square works best),
named by the `logo` field in `apps/web/src/screens/Send.jsx`:

```
bpi.png          bdo.png          metrobank.png    landbank.png
securitybank.png unionbank.png    pnb.png          rcbc.png
chinabank.png    eastwest.png     psbank.png       dbp.png
aub.png          maybank.png      gotyme.png       cimb.png
tonik.png        maya.png
```

Any bank without a file here automatically shows a brand-colored badge instead,
so the app works with or without the images.

Get the official artwork from each bank's press / brand / media kit (or a
licensed logo pack). These marks are the banks' trademarks — make sure you have
the right to use them in your project.
